# Бот для автовладельцев
быстрый старт:
создайте виртуальное окружение:

	python -m venv .venv
 
войдите в него:
Windows:

	.venv/Scripts/activate

Unix-systems:

	source .venv/bin/activate

Установите зависимости:

	pip install -r requirements.txt
